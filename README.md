# 🛡️ Black Wallet - Terminal de Gestão Financeira

**Black Wallet** é um dashboard financeiro desenvolvido em Python com foco em privacidade, simplicidade e uma estética *Cyber/Stealth*. O projeto foi criado para centralizar o controle de fluxo de caixa (ganhos e gastos) com uma interface moderna e sombria (All Black).

---

## 🚀 Funcionalidades

- **Dashboard Cyber:** Interface visual construída com `CustomTkinter` para uma experiência de usuário imersiva.
- **Gestão de Capital:** Registro rápido de ganhos, gastos fixos e variáveis.
- **Análise com IA:** Núcleo de lógica que analisa seu saldo e fornece um parecer sobre sua saúde financeira.
- **Persistência de Dados:** Salvamento automático em arquivo local para garantir a integridade das informações.
- **Segurança Nativa:** Estrutura preparada para proteger informações sensíveis e arquivos de dados pessoais via `.gitignore`.

## 🛠️ Tecnologias Utilizadas

- **Linguagem:** Python 3.13
- **Interface Gráfica:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
- **Manipulação de Imagens:** Pillow (PIL)
- **Compilação:** PyInstaller (para geração de executáveis .exe)
